# Frontend-Fun
Neon Button

## Learnings
To add Neon effect: 
add 3 box shadow of same color but different spread.


```css
  a:hover{
    color:#255784;
    box-shadow: 0 0 10px #2196f3, 0 0 40px #2196f3, 0 0 80px #2196f3;
    }
    /* Box-shadow: horizontal vertical blur color */
```
